# React Hooks Developer Project

This project explores modern React Hooks with a premium UI and clear examples.

## Features
- **useState**: A premium counter that demonstrates state management.
- **useEffect**: 
    - **Window Resize Tracker**: Real-time browser width monitoring.
    - **Live Timer**: Side-effect management with intervals and cleanup.

## Getting Started
1. Run `npm install`
2. Run `npm run dev`
